One-time Setup
To sync data across all devices, this app uses Firebase (free). Follow these steps once, then everyone can log in from any device.
Step 1 — Create a free Firebase project

Go to console.firebase.google.com → Click "Add project" → Name it "Arena Clinic" → Disable Google Analytics → Create project.

Step 2 — Enable Authentication

In your project → Build → Authentication → Get started → Enable Email/Password sign-in method.

Step 3 — Enable Firestore Database

In your project → Build → Firestore Database → Create database → Start in test mode → Choose any region → Done.

Step 4 — Get your app config

Go to Project Settings (⚙️) → "Your apps" → Click "Web" (</>) → Register app → Copy the firebaseConfig values below.

After saving, you'll set up the first Admin account.
Please fill in apiKey, authDomain, and projectId.
